1) Download and install the Vubla module. Detailed installation instructions can be found here http://www.vubla.com/support/magento-installation-guide/.

2) Register your webshop at https://login.vubla.com/?iso=en. 

3) Activate Vubla at https://login.vubla.com/?iso=en. Make sure to enable Vubla in Magento (Detailed instruction can be found here http://www.vubla.com/support/magento-activate-vubla/).
